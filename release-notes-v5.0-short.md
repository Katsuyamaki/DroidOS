# DroidOS v5.0 (Short Release Notes)

DroidOS v5.0 is a major overhaul and the launch of DroidOS Pro.

The Community (free) experience remains strong: all core workflows from v4.0 are still available, plus new v5.0 free improvements.

## Community (Free) - New in v5.0

- Smart Auto Margin + IME sync: launcher margins, tiled windows, and dock/keyboard behavior stay aligned in real time. Apps auto-resize when the keyboard appears to prevent overlap, keyboard visibility follows text-field focus more reliably, and it can remain open for navigation/window control when needed.
- Better full-screen + tiled app coexistence
- Expanded real minimize behavior on supported setups
- Full compatibility for apps that open multiple windows/sub-screens, replacing older package-name-only tracking
- Orientation-aware layouts and profiles
- New broadcast commands + manual re-tile command
- Android 15 compatibility and broad stability fixes

## DroidOS Pro (Expanded capabilities)

v5.0 introduces optional paid feature sets for users who want more control and advanced workflows:

- Customization Module: rename apps/layouts/profiles, Custom Icons, Tab organizer
- Power Module: Dynamic Tiling, Peek Mode, App lock mode, Advanced profile save modes
- Pro License: both modules + all future modules
