# DroidOS v5.0 Release Notes

DroidOS v5.0 is a major platform overhaul and the introduction of DroidOS Pro.

This release expands what DroidOS can do at both levels:
- Community (free) remains fully capable for daily productivity and keeps all core v4.0 functionality.
- DroidOS Pro introduces optional advanced features for users who want deeper control and more automation-focused workflows.

## Community (Free) continues strong

Everything users relied on in v4.0 core workflows remains available in v5.0 Community.

On top of that, v5.0 Community adds major free improvements:

### Smart Auto Margin + IME sync

Launcher margins, tiled windows, and dock/keyboard behavior now stay aligned in real time. Apps automatically resize when the keyboard appears so text input does not overlap or break window layouts. Keyboard visibility follows input focus more reliably, including auto-open/auto-hide behavior, and can remain available for navigation and window control even outside text fields. The result is a more seamless typing and multitasking experience across tiled apps.

### Better full-screen + tiled coexistence

Transitions between normal full-screen apps and tiled workflows are smoother and less disruptive.

### Expanded real minimize

Minimize behavior is improved for supported Samsung DeX and virtual mirror scenarios.

### Smart app window tracking

Full compatibility for apps that open multiple windows/sub-screens, replacing the older package-name-only tracking approach.

### Orientation-aware layouts and profiles

Portrait and landscape behavior can be handled separately with less manual switching.

### New commands and faster recovery

Added broadcast command coverage (including Open & Move To, Open & Swap, Manual Retile) and a manual re-tile shortcut via bubble double-tap.

### Stability and compatibility

Broad bug fixes and Android 15 hardening across launcher/window/typing flows.

## DroidOS Pro in v5.0

v5.0 introduces optional paid expansion features for users who want more advanced control.

### Customization Module

- Rename apps, layouts, and profiles
- Custom Icons (default, standard theme pack, or your own)
- Tab organizer (reorder launcher drawer tabs and toggle tab visibility)

### Power Module

- Dynamic Tiling
- Peek Mode
- App lock mode
- Advanced profile save modes (by mode, layout, apps, or queue)

### Pro License

- Includes both modules
- Includes all future modules
- Available as annual or lifetime
